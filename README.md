# найди-виралку - find-only поиск виралок

Скилл для Claude Code: находит виральные ролики и карусели по теме и критериям и отдаёт список ссылок. И всё. НЕ скачивает, НЕ транскрибирует, НЕ переделывает - только находит и даёт ссылки.

## Установка

Нужен установленный [Claude Code](https://docs.claude.com/claude-code). Затем:

```bash
cd ~/.claude/skills && git clone https://github.com/theivansergeev/naydi-viralku найди-виралку
```

Перезапусти Claude Code - скилл подхватится сам.

Для поиска по YouTube нужен `yt-dlp` (необязательно, без него работает через веб-поиск):
```bash
brew install yt-dlp     # macOS
pip install -U yt-dlp   # Windows / Linux
```

## Как пользоваться

Скажи Claude:
- «Найди виралку про <тема>»
- «Найди виральные ролики по теме <X> за 30 дней»
- «Поиск виралок по критериям» (площадка, мин. просмотры, свежесть, формат)

Получишь ранжированный список ссылок с метриками. Без скачивания и без личных cookie.
